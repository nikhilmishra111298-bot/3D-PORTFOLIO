Content Broadcasting System

Content Broadcasting System is a role-based web application where teachers can upload content, principals can approve or reject it, and approved content can be displayed publicly. The project focuses on clean UI, approval workflow, dashboard cards, and content status management.

Tech Stack

Key Features

  • Teacher dashboard with uploaded content status
  • Principal approval and rejection workflow
  • Pending, approved, and rejected content tracking
  • Public live content display page
  • Responsive frontend design
  • Reusable UI components
